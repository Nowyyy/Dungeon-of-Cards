Permet de déplacer d'option en option le rectangle de sélection dans le menu charger.

Paramètres
*etatpermet de changer l'affichage selon l'écran dans lequel se trouve le jeu
charger_rectle rectangle contenant le texte relatif à la sauvegarde
retour_rectle rectangle contenant le texte "retour", pour retourner au menu principal
**rect_selle rectangle de sélection permettant de sélectionner une option
*sounds[NB_SON]le tableau contenant les sons
Renvoie
retourne False pour fermer la fenetre, True pour la garder ouverte
